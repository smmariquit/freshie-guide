# Contributing to Freshie Guide

Thanks for helping incoming UPLB freshmen find accurate, welcoming guides.

## Report issues

Open a [GitHub issue](https://github.com/smmariquit/freshie-guide/issues) when a handbook page is outdated, unclear, or wrong. Include the page URL and what should change.

## Developers

### Setup

```sh
bun install   # or npm install
bun dev       # http://localhost:4321
```

### Pull requests

1. Branch from **`main`**.
2. Use [Conventional Commits](https://www.conventionalcommits.org/) (`fix(guide): …`, `content: …`).
3. Open a PR to **`main`**; link the issue when applicable.
4. Prefer verifiable campus info (official pages, org contacts). Do not commit secrets.
